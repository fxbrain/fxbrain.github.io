Javascript Signed Integer Conversions

Written by Stefan Schwarz

Many folksd are confused on how asm.js is using specific operators to force values to become signed integers. This article tries to shed a light upon this affair.

We can dig into SpiderMonkey using gdb to realise the type conversion life:

gdb --args js
[...]
(gdb) b JS::ToInt32
(gdb) r

js> var i = 123.3;
js> i|3

Breakpoint 2, JS::ToInt32 (cx=0x7ffff6b67180, v=$jsval(123.3), out=0x7fffffffda60)
at ../../../dist/include/js/Conversions.h:149

So we proofed that the ‘|’ operator does type coercion into signed integer.
